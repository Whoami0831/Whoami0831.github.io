HTML5

基础知识

  • br 换行
  • hr 水平线
  • strong 粗体
  • em 斜体

行内元素和块元素

  • 块元素
    • 无论元素多少，该元素独占一行
    • p，h…
  • 行内元素
    • 内容撑开宽度，左右都是行内元素的可以排在一行
    • a,strong,em…

列表标签

  • 无序列表
  • 有序列表
  • 定义列表

表格标签

媒体元素

  • 视频元素 video
  • 音频元素 audio

页面结构分析

元素名 描述
header 标题头部区域的内容
footer 标记脚部区域的内容
section Web页面中的一块独立区域 网页主题
article 独立的文章内容
aside 相关内容或应用
nav 导航类辅助内容

iframe内联框架

在一个网页中嵌套另外一个网页

表单post和get提交

  • form
  • action 表单提交的位置，可以是网站，也可以是一个请求处理地址
  • method 提交方式 post或get
    • get方式提交：可以在URL中看到提交的信息，不安全，高效
    • post：比较安全，传输大文件
  • input 文本输入框
    • value：默认初始值
    • maxlength：最大长度
    • size：文本框长度
    • radio 单选框
    • checkbox多选框 checked 默认选中
    • button按钮
    • file 文件域
    • email 邮件验证

下拉框，列表框

  • selected 默认被选择
  • textarea文本域
  • select下拉框

搜索框滑块和简单验证

邮件验证，url验证,数字验证,滑块,搜索框

表单的应用

隐藏域 hidden

只读 readonly

禁用 disabled

增强鼠标可用性 label for

表单初级验证

为了减轻压力，保证安全性
